NclBool nclStartEcgStream ( int  nymiHandle)

Starts a stream of ECG samples, with a rate of 250Hz. The user should put a finger from the opposing hand on the top metal piece of the Nymi to measure their ECG.

Precondition
NCL is initialized. The specified Nymi is validated and has a free command channel.
Postcondition
The Nymi's command channel is taken. An NCL_EVENT_ECG_START will be triggered when the command channel is free, and then NCL_EVENT_ECG events will be triggered as the Nymi streams ECG.
Parameters
[in]nymiHandleThe handle of the Nymi whose ECG stream you want to start.
Returns
NCL_TRUE On successful initiation.
NCL_FALSE On failed initiation.
NclBool nclStopEcgStream ( int  nymiHandle)

Stops the stream of ECG samples.

Precondition
NCL is initialized. The specified Nymi is validated and has a free command channel.
Postcondition
The Nymi's command channel is taken. An NCL_EVENT_ECG_STOP will be triggered when the command channel is free. At this point no more NCL_EVENT_ECG events will be triggered.
Parameters
[in]nymiHandleThe handle of the Nymi whose ECG stream you want to stop.
Returns
NCL_TRUE On successful initiation.
NCL_FALSE On failed initiation.
